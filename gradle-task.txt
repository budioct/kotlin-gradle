perintah task gradle di windows

// melihat semua task di gradle
./gradlew tasks --all

// menjalanakan gradle
./gradlew run

// menjalankan kompilasi gradle
./gradlew classes

// menjalankan kompilasi gradle unit test
./gradlew test


// di gradle pusat untuk mengatur dependency dan costum task di file
name-project\app\appbuild.gradle.kts

// untuk mengatur module project gradle di file
name-project\settings.gradle.kts

// untuk membuat distribution file di gradle
gradle distTar
gradle distZip

=> hasil distribution file bisa kita lihat di direktori: app/build/distributions

// menjalankan file distribution yang telah di buat
// windows
kotlin-gradle/app/build/distributions/app/bin/app.bat

// linux atau Mac
kotlin-gradle/app/build/distributions/app/bin/app
